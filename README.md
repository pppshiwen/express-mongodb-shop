# express-mongodb-shop
商品管理系统
cd到文件根目录


cd express-mongodg-shop
```

```
npm install 或 cnpm install(个人比较喜欢使用后者，下载依赖模块速度较快)
```

```

最后开启后台服务

```
node app2.js
```
在浏览器输入http://localhost:3000/admin/
